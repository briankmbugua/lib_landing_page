Library Management System

This system allows you to manage your library books, subjects, and users.

View the GitHub repository

Features

Books

Manage your library books, including adding, editing, and deleting books.

Subjects

Manage your library subjects, including adding, editing, and deleting subjects.

Users

Manage your library users, including adding, editing, and deleting users.

About

This project was inspired by my love of books and my desire to create a system that would make it easier for people to manage their library collections. I started working on this project in January 2023, and I am currently working on adding new features and improving the user experience.

This project was created as a portfolio project for Holberton School. I am a software engineer student at Holberton School, and I am passionate about building software that makes a difference in the world.
